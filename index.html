Webseiten mit HTML und CSS gestalten

HTML

HTML ist eine (Text-)Auszeichnungssprache, die keine Befehle, sondern Tags verwendet. Webseiten werden "geschrieben", um dann von einem entsprechenden Programm, dem Browser, "gelesen" zu werden.

HTML strukturiert eine Webseite und wird "geschrieben" und nicht "programmiert". Sie schreiben "Quelltext" oder "Markup". HTML besteht aus Elementen. Dazu werden Tags verwendet, die Attribute mit Werten haben können:
<element attribut="wert">Mein Text</element>

in Element kann mehrere Attribute besitzen, die in beliebiger Reihenfolge im Starttag stehen. Die Attribute ergänzen das HTML-Element durch konkretere Eigenschaften und können verschiedene Werte haben.

CSS

CSS steht für Cascading Stylesheets, einen einfachen Mechanismus, um Formatierungen für Webseiten zu schreiben.

Während HTML eine Webseite lediglich strukturiert, gestaltet CSS diese über eine Sammlung von Formatvorlagen, dem so genannten "Stylesheet". Alle HTML-Elemente können Sie mit CSS formatieren.

Stylesheets können in einem style-Element, das sich im head-Element des HTML-Dokuments befindet, aus einer .css-Datei eingebunden oder direkt definiert werden. Dazu werden Formatanweisungen verwendet, die aus einem Selektor (z.B. code) und einem Deklarationsblock in geschweiften Klammern (z.B. { font-size: 105%; }) bestehen.

Eine Deklaration besteht aus einer Eigenschaft (z.B. font-size), einem Doppelpunkt und einem Wert (z.B. 105%), gefolgt von einem Semikolon: code { font-size: 105%; }

Mit den folgenden Anweisungen im head-Element legen Sie für den gesamten Text im Dokument die Schriftart und die Schriftfarbe fest:
<style>
 body{
  font-family: Arial;
  color: black;
 }
</style>
